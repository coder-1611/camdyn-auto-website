# Camdyn Auto Repair — Website Redesign

Redesign of [camdynautorepair.com](https://camdynautorepair.com) (family-owned auto repair shop in Round Rock, TX), built in Claude Design.

Pages: `index`, `services`, `about`, `savings`, `reviews`, `contact` (`*.dc.html`), with shared `SiteNav` / `PageHeader` / `SiteFooter` components. `support.js` is the dc-runtime that renders the `.dc.html` component format; `fx.js` is the scroll-reveal/parallax motion system. Light/dark theme toggle persisted via `localStorage`.

Source: Claude Design project "Camdyn Auto Repair redesign".
